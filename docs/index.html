Installation

Composer (Recommended)

Create or update your composer.json

{
    "require": {
        "kzykhys/ciconia": "~1.0.0"
    }
}

$ php composer.phar update

Git

$ git clone https://github.com/kzykhys/Ciconia.git
$ cd Ciconia
$ php composer.phar install

Phar

$ curl -O https://github.com/releases/1.0.0/ciconia.phar
