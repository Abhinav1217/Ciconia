Ciconia

A New Markdown Parser for #PHP

The Markdown parser for PHP5.4, it is fully extensible.
Ciconia is the collection of extension, so you can replace, add or remove each parsing mechanism.

Download Now Fork on Github

You can install Ciconia via Composer

$ vi composer.json
{
  "require": {
    "kzykhys/ciconia":"~1.0.0"
  }
}
$ curl -s http://getcomposer.com/installer | php
$ php composer.phar install

Quick Start

As a library

<?php

require __DIR__ . '/vendor/autoload.php';

$ciconia = new \Ciconia\Ciconia();
$ciconia->render('Markdown is **awesome**');
<p>Markdown is <strong>awesome</strong></p>

As a command-line tool

bin/ciconia /path/to/file.md > output.html
php ciconia.phar /path/to/file.md > output.html
